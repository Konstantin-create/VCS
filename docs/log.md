# Log command

Print last commit log
```shell
~ python vcs.py log
Commit: 39458c22432915f28ca4f99e5fa29a8e7b9b65c74d5b3a4c76b86de24f82ee1b
Message: Initial
Time stamp: 2022-08-09 18:06:39.970791
Parent: None
```

## Help message
```shell
vcs log - Print last commit log
```
